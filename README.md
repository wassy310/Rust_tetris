# Rust_tetris
## Overview
Tetris implemented in Rust.  
It also serves as a study of Rust.  
You can learn the following points  
- computation
- Branching, Iteration
- Variables, Arrays
- Functions
- String output

## Getting Started
- Clone this project.
  - `git clone https://github.com/wassy310/Rust_tetris`
- Debug & Run
  - `cargo run`
